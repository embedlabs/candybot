Detailed Description

The RSSItem class models a feed item. The class holds information about the RSSItem including:

  • title,
  • image,
  • link and
  • description properties.

The title should always be present but the image, link and description properties may be null.

An RSSItem may be queried even if it is marked as having been read before.
