RSSFeed.RequestNextItemCanceledException Class Reference

List of all members.

Detailed Description

This exception will arise when the RSSFeed.cancelRequestNextItem() method is called.
The exception will be passed to the RSSFeedObserver.feedDidFailToReceiveNextItem(RSSFeed, Exception) callback method.
