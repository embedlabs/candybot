Detailed Description

The SocialProviderController is used to connect a user to a specific social network, (modelled by SocialProvider ).

Scoreloop supports the following social networks:

  • MySpace
  • Twitter
  • Facebook

Basic usage:

  1. Get an instance of SocialProvider.
  2. Use one of the getSocialProviderController methods, below, to create an instance of the controller.
  3. Call connect(android.app.Activity) to connect the user to the social network.

Note that the connection process will most likely open a view for authentication purposes, (e.g. the user may have to enter their username and password to access their social network).

See also:
SocialProvider, SocialProviderControllerObserver

final void connect ( Activity  context)

Starts the connection process. This method will try to connect the session user to the SocialProvider controlled by the SocialProviderController.

The call is only valid, if the user is not already connected, and provided that the user already has a login name.

IMPORTANT: It is likely that this method will display some authorization dialogs from the social network in question.
