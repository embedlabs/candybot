Detailed Description

The ActivitiesController retrieves lists of activities from the server. The list that is retrieved may relate to:

  • a user's activities
  • a game's activities
  • buddies/friends activities
  • all activities associated with the current game
  • community related activities (across all games and all users)

An example activity might contain the message: "Player001 has scored 50 points in SLDemoCore".

Basic usage:

  1. Get an instance of ActivitiesController or re-use an existing one.
  2. Use one of the load methods:
    • loadActivitiesForGame()
    • loadActivitiesForUser()
    • loadBuddyActivities()
    • loadCommunityActivities() or
    • loadGameActivities() to make the request to the server.
  3. Await notification of success via delegate callbacks.
  4. Use getActivities() to access the list that was retrieved.

Member Function Documentation

List<Activity> getActivities ( )

Accessor that returns the list of activities retrieved from the server by either loadBuddyActivities() or loadGameActivities().

If this method is called before a request to the server has been made, an empty list will be returned.

Returns:
java.util.List of Activity objects, or an empty list.

Requests all activities associated with the specified game. All the activities of the game across all the users will be returned.
