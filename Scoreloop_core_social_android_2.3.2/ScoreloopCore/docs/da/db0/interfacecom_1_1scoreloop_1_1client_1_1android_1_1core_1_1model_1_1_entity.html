Detailed Description

All model objects that have a Scoreloop-assigned identifier from this Entity interface, including:

  • Achievement
  • Activity
  • Challenge
  • Game
  • Score
  • User

Member Function Documentation

String getIdentifier ( )

Returns the identifier of the entity. The identifier is generated and assigned by the Scoreloop servers. All entities retrieved from the server will automatically have this property set. Any entities that are created locally, (such as Score), will not have an identifier assigned to them until they have been first submitted to Scoreloop.
