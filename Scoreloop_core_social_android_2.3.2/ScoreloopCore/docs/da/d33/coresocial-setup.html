Project Set-up

Android.com recommends using the open source Eclipse IDE with the Android ADT plugin as the preferred development environment for producing Android applications.
Consequently, the following instructions are specific to setting up Scoreloop CoreSocial with Eclipse. However many of the steps are also applicable generally.
Please visit http://developer.android.com/guide/developing/other-ide.html for instructions on how to set up Android with other development environments. In order to use the CoreSocial SDK you should have already be using:

  • Android 1.5 or higher
  • Java 1.6

Step-by-step Project Set-up

1. Go to https://developer.scoreloop.com and download the latest version of the CoreSocial SDK.

2. Unpack the Scoreloop download package. The following command shows an example of how to do this:

$LOCAL_DOWNLOAD_LOCATION>tar -xzf Scoreloop_core_social_android_2.3.2.tar.gz

3. The unpacked contents of the file are as follows:

  • BrandingGuidelines.pdf, (instructions on how to use Scoreloop icon badges in your application),
  • brandingAssets.tar.gz, (Scoreloop icon badges),
  • Docs, (documentation directory for ScoreloopUI),
  • LicenseAgreement.pdf, (the Scoreloop license agreement),
  • SLDemoCore, (a sample Scoreloop application)
  • SLDemoUI, (a sample Scoreloop application with user interface),
  • ScoreloopCore, (contains the ScoreloopCore JAR file and the documentation for using the CoreSocial API's),
  • ScoreloopUI, (Scoreloop integration with a ready-built interface),
  • readme_SDK_CORE_apichanges_*.txt, (API changelogs),
  • readme_SDK_CoreSocial_releasenotes_2.3.2.txt, (a README).

4. When setting up your project to use CoreSocial it is already assumed that:

  • you have already downloaded and installed an appropriate IDE.
    (Eclipse is recommended. Go to http://www.eclipse.org for details).
  • you have already downloaded and installed the Android SDK and configured your IDE for Android development.
    (Go to http://developer.android.com for more details). Note that you will need Android 1.5 or higher and Java 1.6.

To start using CoreSocial, the CoreSocial JAR must be added to the Java Build Path of your project. Open your game project in Eclipse and select:

   Project -> Properties -> Libraries -> Add external JAR

This will open a browse dialog on your machine. You should navigate to the CoreSocial JAR file which is located at
{$CoreSocialDownloadLocation}/ScoreloopCore/jar/scoreloop-core.jar. Select the scoreloop-core.jar file and click ok.


6. The next step is to go to https://developer.scoreloop.com and register your game. Once the game has been registered, navigate to the
Configure page of the website and note the Game ID and Game Secret. These are both required
as part of the initialization process described below.
On the Configure page of the website also note the "Get scoreloop.properties" link. You should download this file to a safe location as it is required as part of the project setup.

7. Locate the scoreloop.properties file you downloaded in Step 6 and add this to the /assets folder of your game project.

Once you are done with the Project Set Up, continue with Initialization
