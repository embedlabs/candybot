Detailed Description

The delegate protocol for the ChallengeController. The ChallengeControllerObserver will receive callback notifications from the server in cases where:

  • the challenge cannot take place due to the user having an insufficient amount of currency to cover the challenge stake,
  • if the request to accept the challenge fails,
  • if the request to reject the challenge fails.
