CoreSocial SDK and the Scoreloop API Server

CoreSocial SDK is a set of APIs that is implemented on client devices and is the basis for integrating Scoreloopâ€™s social features into game applications. The SDK talks to the Scoreloop API Server, running on a high availability server farm maintained at 24x7 by the Scoreloop operations team.

Scoreloop has its roots in providing state-of-the-art server-side technology. The security we employ to ensure the integrity of our platform comes from our expertise in high volume server infrastructure deployment and maintenance. Beyond security, our servers themselves are completely technology agnostic. This means we can connect to anything that has internet access - be that an iPhone, iPad, or Wii.

Information such as user details and game scores are sent from client devices to the server. The server then manages things like updating user profiles, score comparisons and score sorting. Communication between the clients and servers is managed by a set of data controllers.
