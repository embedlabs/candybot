Welcome Back Toast

ScoreloopUI provides an option to show a "welcome back" toast that displays the user's name when they return to the game.
The Welcome Back Toast

To show this welcome back toast, you need to call the ScoreloopManager.showWelcomeBackToast() method. This method takes an argument of type long which is the time interval in milliseconds after which the toast should be shown. 
 long delay = 2000;
 ScoreloopManagerSingleton.get().showWelcomeBackToast(delay);
