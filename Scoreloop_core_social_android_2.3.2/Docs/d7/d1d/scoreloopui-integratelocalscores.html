Local Scores

If a network error occurs when a score is submitted to Scoreloop, the score is submitted to the Local Leaderboard. In this case the return code is "STATUS_SUCCESS_LOCAL_SCORE = 2".
You can also specify explicitly that a score is submitted to the local (offline) leaderboard only.
Please note that in neither case the score will not be automatically submitted to Scoreloop when the user regains network connectivity.
The local Scores can be submitted to Scoreloop by calling the method submitLocalScores(Runnable continuation) or through the Local Leaderboard View (ie through the Button - "Submit Scores to Scoreloop"). Please note that the Button would disappear once the local scores have been synchronized with the Scoreloop Server.

The Local Leaderboard View looks like this (please note the "Submit Scores to Scoreloop" button in the figure):

Once the local score has been successfully submitted to Scoreloop, the Local Leaderboard View looks like this:

The Local Leaderboard View

The sample code snippet below shows how to submit a score to the local leaderboard: 

  //create the score object and set the components 
 final Score score = new Score((double) scoreResult, null);
 score.setMode(mode);
 
 bool submitLocallyOnly = true;
 
 //submit the score
 ScoreloopManagerSingleton.get().onGamePlayEnded(score, submitLocallyOnly);

The code example below shows how to invoke the local leaderboard Screen:

 final Button leaderboardsButton = (Button) findViewById(R.id.button_leaderboards);
 leaderboardsButton.setOnClickListener(new View.OnClickListener() 
 {
        @Override
        public void onClick(final View v) {
 
                final Intent intent = new Intent(this, LeaderboardsScreenActivity.class);
 
                // specify the leaderboard that will open by default 
                intent.putExtra(LeaderboardsScreenActivity.LEADERBOARD, LeaderboardsScreenActivity.LEADERBOARD_LOCAL);
 
                startActivity(intent);
        }
 });
