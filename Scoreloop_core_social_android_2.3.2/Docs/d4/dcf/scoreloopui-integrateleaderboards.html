Leaderboards

The Leaderboard view is the recommended entry point into Scoreloop if your game supports scores and leaderboards only.

The Leaderboards View

Explicit invocation of the Leaderboards View

The code example below shows how to open the leaderboard view from your game code. You could include a link to this screen from from your game's main menu.

 //You should replace button_leaderboards with the appropriate id name from your project
 final Button leaderboardsButton = (Button) findViewById(R.id.button_leaderboards);

//Set an OnClickListener 
leaderboardsButton.setOnClickListener(new View.OnClickListener() {
      
       @Override
       public void onClick(final View v) {
      
            //When the button is clicked, the LeaderboardsScreenActivity will be launched.
            startActivity(new Intent(this, LeaderboardsScreenActivity.class));
       
       }
 });
