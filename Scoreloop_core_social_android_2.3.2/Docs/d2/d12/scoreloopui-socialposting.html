Posting Scores to Social Networks

Once you have submitted the score to Scoreloop, as outlined in the Score Submission Integration Guide, you may wish to give the user the option of posting their score to a social network like:

  • MySpace
  • Facebook
  • Twitter

Displaying the dialog for posting to these networks does not happen automatically but must be invoked explicitly. A suitable place where you might like to do this is within the OnScoreSubmitObserver.onScoreSubmit() callback. The example below shows a two-step process:

  • The callback method displays a dialog indicating that the score has been uploaded/failed.
  • The method that is called when the message dialog is dismissed overlays the score posting dialog, (provided the score was uploaded successfully). 

public void onScoreSubmit(final int status, final Exception error) {

// This lets the player know the outcome of the score submission
// attempt, (score submitted, challenge result, challenge balance error, network error).
// This example uses startActivityForResult as it notifies us when the
// overlay finishes.
 
    startActivityForResult(new Intent(this, ShowResultOverlayActivity.class), SHOW_RESULT);
}

The onActivityResult method is called when the message dialog shown above is dismissed. It is from here that we invoke the PostScoreOverlayActivity that manages posting to social networks. 

@Override
protected void onActivityResult(final int requestCode, final int resultCode, final Intent data) {

      switch (requestCode) {

              case SHOW_RESULT:
                     if (_submitStatus != ScoreloopManager.STATUS_ERROR_NETWORK) {
                        // Show the post-score activity unless there has been a network error.
                        startActivityForResult(new Intent(this, PostScoreOverlayActivity.class), POST_SCORE);
                     } else { 
   
                        finish();
                     }
               
                     break;

             case POST_SCORE:
             
                        // Here we get notified that the PostScoreOverlay has finished.
                        // in this example this simply means that we're ready to return to the main activity
                        finish();
                        break;
             default:
                        break;
     }
  }

Make sure to add the following line to the AndroidManifest.xml: 

<activity android:name="com.scoreloop.client.android.ui.PostScoreOverlayActivity" android:theme="@style/sl_dialog"/>
