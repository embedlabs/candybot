The Buddies View

Explicit Invocation of the Buddies View

The buddies view is always accessible from the "shortcuts" panel that is present at the bottom of every
ScoreloopUI screen, so it is unlikely that you will need to explicitly call it from elsewhere in your game.
However, for those who wish to do this the example code below shows how you might go about doing this. 

//You should replace friends_button with the appropriate id name from your project
final Button friendsButton = (Button) findViewById(R.id.button_friends);

friendsButton.setOnClickListener(new View.OnClickListener() {

       @Override
       public void onClick(final View v) {

       startActivity(new Intent(this, BuddyScreenActivity.class));

       }
});

You should make sure that the Android Manifest is modified so that it references the BuddiesScreenActivity
by adding the following line to it:

<activity android:name="com.scoreloop.client.android.ui.BuddiesScreenActivity" android:theme="@android:style/Theme.Black.NoTitleBar.Fullscreen"/>
