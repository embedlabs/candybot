The Profile View

Explicit Invocation of the Profile View

The Profile view is directly accessible from the Scoreloop Entry Screen, so in most cases it will be unnecessary to explicitly
launch the view from elsewhere in their games. However, for those who require this, the code example below shows how to
go about doing this. 

 //You should replace button_profile with the appropriate id name from your project
 final Button profileButton = (Button) findViewById(R.id.button_profile);
 
 profileButton.setOnClickListener(new View.OnClickListener() {
        @Override
        public void onClick(final View v) {
                startActivity(new Intent(this, ProfileScreenActivity.class));
        }
 });

You will also need to update the Android Manifest so that it references the ProfileScreenActivity. Add the following line
to the AndroidManifest.xml in your project:

 <activity android:name="com.scoreloop.client.android.ui.ProfileScreenActivity" android:theme="@android:style/Theme.Black.NoTitleBar.Fullscreen"/>
