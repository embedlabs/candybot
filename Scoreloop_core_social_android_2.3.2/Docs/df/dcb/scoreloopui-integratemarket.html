Social Market

ScoreloopUI also provides a Social Market View that allows you to display the following:
  • Featured Games
  • Popular Games
  • New Games and
  • User ºs buddies Games
The Social Market View

Note: The Social Market view is always accessible from the "shortcuts" panel that is present at the bottom of every ScoreloopUI screen, so it is unlikely that you will need to explicitly call it from elsewhere in your game. However, for those who wish to do this the example code below shows how you might go about doing this. 
//You should replace button_social_market with the appropriate id name from your project
final Button socialMarketButton = (Button) findViewById(R.id.button_social_market);

socialMarketButton.setOnClickListener(new View.OnClickListener() {
    
       @Override
       public void onClick(final View v) {

            startActivity(new Intent(this, SocialMarketScreenActivity.class));

       }
});

Important addition to AndroidManifest.xml

If you are integrating the social market feature, you will need to update the Android Manifest so that it references the SocialMarketScreenActivity. Add the following line to AndroidManifest.xml in your project: 

<activity android:name="com.scoreloop.client.android.ui.SocialMarketScreenActivity" android:theme="@android:style/Theme.Black.NoTitleBar.Fullscreen"/>
