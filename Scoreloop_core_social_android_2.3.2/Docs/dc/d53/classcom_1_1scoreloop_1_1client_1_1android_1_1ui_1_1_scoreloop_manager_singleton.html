Detailed Description

The ScoreloopManagerSingleton class is used to create and retrieve the singleton instance of ScoreloopManager that is shared by all ScoreloopUI activities.

Basic usage:

  1. Create an instance of ScoreloopManager using init(final Context).
  2. Access the instance by calling get().
