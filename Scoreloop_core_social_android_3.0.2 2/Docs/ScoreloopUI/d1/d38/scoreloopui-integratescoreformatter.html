ScoreFormatter

The Scoreformatter allows you to configure how the score object should be displayed in ScoreloopUI. The configuration is done in the "scoreloop.properties" file via "format.score".

The following properties define the elements of a score to be shown in the given ScoreloopUI screen. e.g. show score and level on the leaderboards view or show level and mode on the challenges view. 

 ui.format.score.leaderboard = ScoresAndLevelFormat
 ui.format.score.challenges = LevelAndModeFormat
 ui.format.score.socialnetworkpost = DefaultFormat

The example snippet below how you could modify to display "the score and the level" on all the views. 

 format.score = ScoresAndLevelFormat="%r\n%l";
 ui.format.score.leaderboard = ScoresAndLevelFormat;
 ui.format.score.challenges = ScoresAndLevelFormat;
 ui.format.score.socialnetwork = ScoresAndLevelFormat;

The example snippet below how you could modify to display "the score result as time in hh:mm:ss format" on the Leaderboard Screen. 

 ui.format.score = DefaultFormat="%T";ResultTimeConversion="%h:%0m:%0s";
 ui.format.score.leaderboard = DefaultFormat;

The example snippet below how you could modify to display "the score result as time in hh:mm:ss format" on the Leaderboard Screen and "the level and Mode" in the challenges history of a player. 

 ui.format.score = DefaultFormat="%T";ResultTimeConversion="%h:%0m:%0s";LevelAndModeFormat="%l %m\n%T"
 ui.format.score.leaderboard = DefaultFormat; 
 ui.format.score.challenges = LevelAndModeFormat; 
 ui.format.score.socialnetwork = DefaultFormat;

The example snippet below how you could modify to display different modes as score based and time based. 

 ui.format.score = LevelAndModeFormat="%r %t";ModeSymbol=;mode:0=Total score;mode:1=Best time; 
 ui.format.score.leaderboard = LevelAndModeFormat;

The example snippet below how you could modify to display the score "result" and "minor result" as "double values". 

 ui.format.score = ScoresAndLevelFormat="r:%.3r q:%.3q"
 ui.format.score.leaderboard = ScoresAndLevelFormat;

For a full specification, please refer to the ScoreloopCore's ScoreFormatter documentation.
