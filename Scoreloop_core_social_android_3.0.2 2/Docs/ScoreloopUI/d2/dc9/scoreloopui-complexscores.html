Using Complex Scores

A complex score is represented by the CoreSocial SDK Score class. An advanced integration of ScoreloopUI lets you submit complex scores as part of your game.The complex score can contain:

  • the main score result (mandatory),
  • a minor result (optional),
  • the 'level' at which the score was obtained (optional),
  • the 'mode' - gameplay setting at which the score was achieved (optional),
  • the 'score context' - additional game related data that you want to associate with the score object (optional).

The result, minor result, and level can be used in combination for score comparison purposes. The score mode is used in the generation of separate leaderboards for each gameplay setting in your game.

Submitting a complex score:

1. Create the Score object with the following constructor:

//The Double argument passed is the main game result, the Map
// argument is a hash used to pass an additional score context,
// (which could be some strings associated with the score, for example).
// The main game result cannot be null, (although it can be zero). 
// The score context may be null.

Score(final java.lang.Double, final java.util.Map<String, Object>)

//Example as it might appear in your code
Score myComplexScore = new Score(new Double(12), null);

2. Set/modify the score result, minor result, level, mode, or context by using the following APIs:

myComplexScore.setResult(final java.lang.Double); 

myComplexScore.setMinorResult(final java.lang.Double);

myComplexScore.setLevel(final java.lang.Integer);

myComplexScore.setMode(final java.lang.Integer);

myComplexScore.setContext(java.util.Map<String, Object>);

3. Access the score data using the following APIs:

myComplexScore.getResult(); 

myComplexScore.getMinorResult();

myComplexScore.getLevel();

myComplexScore.getMode();

myComplexScore.getContext();

4. To submit the score, call the following method ScoreloopManager.onGamePlayEnded(Score, Boolean) from the class that you have written implementing the OnScoreSubmitObserver.

ScoreloopManagerSingleton.get().onGamePlayEnded(myComplexScore, false);
