Detailed Description

The OnStartGamePlayRequestObserver receives notifications and callbacks from the server when a Scoreloop challenge is started or accepted.

The observer must be correctly set in the ScoreloopManager if the game supports challenges.

Basic Usage:

  1. Implement the onStartGamePlayRequest(Integer) method.
  2. Set the observer by calling ScoreloopManager.setOnStartGamePlayRequestObserver(OnStartGamePlayRequestObserver).
