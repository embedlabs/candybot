Awards and Achievements

Q. What is an award?
You can use awards to reward players of your game with an amount of in-game currency when they complete a certain task or attain a specific goal.

Q. How do I configure awards for my game?
You will first have to go to https://developer.scoreloop.com and navigate to the 'Awards' page. There you can enter the details for your awards. When the award list is complete you will need to download the awards bundle from the same page. This file, sl_award_bundle.zip, needs to up unzipped and the resulting folder added to the /assets directory of your game project. This provides local access to awards in your game application.

Q. Can I configure an award so that it is reached in stages?
Yes. You can configure an award so that it is reached in stages. This is done on https://developer.scoreloop.com . To do so, you should define the 'required count' property on the 'Create New Award' page equal to the number of stages required before the award is achieved. For example, setting this property equal to 10 would mean that the award is achieved in 10 incremental stages. In the gameplay each 'stage' could correspond to finding a secret jewel, for example.

Q. What is the difference between an award and an achievement?
Awards and achievements are linked concepts in Scoreloop. An award refers to the reward a user achieves by reaching a certain goal. An 'Achievement' is a CoreSocial SDK API that models whether an award has been achieved by a user or not. See the CoreSocial SDK documentation for more details.

Q. How do I set up awards in my game application?
You need to configure awards on https://developer.scoreloop.com and download the awards bundle that is generated there. This bundle must be added to your game project. For more details on how to set up awards in your game application, please refer to the Awards and Achievements in the advanced integration section.
