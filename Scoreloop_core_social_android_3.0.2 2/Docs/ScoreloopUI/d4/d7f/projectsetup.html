Project Set-up

Step by Step Project Set Up

1. Go to https://developer.scoreloop.com and download the latest version of the CoreSocial SDK for Android.

2. Unpack the Scoreloop download package. The following command shows an example of how to do this:

 $LOCAL_DOWNLOAD_LOCATION>tar -xzf Scoreloop_core_social_android_3.0.tar.gz

3. The unpacked contents of the file are as follows:

  • Readme.txt, (a README that you should be looking at first),
  • Releasenotes_2.4.1.txt, (the release notes for this release),
  • Docs, (ScoreloopUI and ScoreloopCore docs. Also contains API change History),
  • ScoreloopUI, (Scoreloop integration with a ready-built interface),
  • SLDemoUI, (a sample Scoreloop application with user interface),
  • ScoreloopCore, (contains the ScoreloopCore JAR file),
  • SLDemoCore, (a sample Scoreloop application),
  • BrandingGuidelines.pdf, (instructions on how to use Scoreloop icon badges in your application),
  • brandingAssets.tar.gz, (Scoreloop icon badges),
  • LicenseAgreement.pdf, (the Scoreloop license agreement).

4. Import the ScoreloopUI project into your IDE. In Eclipse do the following and then navigate to the location of the ScoreloopUI project on your machine: 

 File -> Import -> Existing Projects Into Workspace

5. ScoreloopUI must be added to your game project as a 'Library Project'. To do this:

  1. Right-click on your game project in Eclipse.
  2. Select 'Properties'.
  3. Select 'Android'
  4. Select the 'Add' button in the 'Library' section.
  5. Locate the ScoreloopUI and click it.




6. Make sure that the scoreloop-core.jar is added to the Java Build Path of your project. Navigate to your project -> properties -> Libraries and check if scoreloop-core.jar is present in Properties -> Java Build Path -> Libraries. If not, add it by selecting:

Project -> Properties -> Java Build Path -> Libraries -> Add external JAR

This will open a browse dialog on your machine. You should navigate to the scoreloop-core.jar file which is located at {$DownloadLocation}/ScoreloopCore/jar/scoreloop-core.jar and Select the scoreloop-core.jar file and click ok.

7. The next step is to configure your game properties. Please navigate to Configuring Your Game Properties to find out how to do this.

Troubleshooting

Sometimes it is possible to encounter some issues when settup up your project to use ScoreloopUI.

General tip
A general tip for resolving erroneous, non-specific errors when using ScoreloopUI is to perform a "project clean". You can do this in two ways:

  • From the Eclipse IDE menubar by selecting:
    • Project -> Clean
  • Launching the Eclipse IDE from the commandline in "clean" mode:
    • $command_prompt>/$PATH_TO_ECLIPSE/eclipse -clean

Issue: My project has a lot of errors saying that Scoreloop classes cannot be resolved to a type
The most common cause of this is forgetting to add the scoreloop-core.jar to your project. To resolve this:

  • Make sure you have followed Step 6 in the setup instructions, above.
  • Carry out a project clean.
  • Rebuild the project.

Issue: I keep adding the ScoreloopUI as a library project as per the instructions, but it doesn't seem to be working.
Sometimes artefacts in the Eclipse workspace can cause problems when adding library projects. To resolve this issue:

  • Launch a "clean start" to Eclipse from the commandline.
  • Add ScoreloopUI as a library project as per step 5 above.
  • Rebuild your project.
