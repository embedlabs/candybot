News

The News View

The ScoreloopUI "News" feature lets you stream updates relating to Scoreloop gaming activities and/or updates from an
external RSS feed directly to your game. Integrating this feature consists of two steps:

  1. Configuring the RSS feed on https://developer.scoreloop.com
  2. Updating the scoreloop.properties file to enable the "News" feature. The steps below show how to do this.

Configuring the Feed on https://developer.scoreloop.com

1. Go to https://developer.scoreloop.com and navigate to the "Feed" page. Click the "Create new news item" button.

2. Enter the details for the news feed in the fields. Tick the isActive box to activate the feed. Tick the isSticky checkbox,
to make the item a "sticky item. This means that it will still be displayed even after it has been read.

3. Choose the feed method. You can choose to only display news items, (that is items that originate from one of your configured
feeds), or you can choose to display both news items and items relating to Scoreloop activities, (such as winning challenges, or
achieving an award).

4. Repeat Steps 1 to 3 to configure additional feeds.

Modifying the scoreloop.properties file

Modify the scoreloop.properties file so that the "News" feature is enabled. Make sure that it includes the following line:

ui.feature.news = true

You will also need to update your Android Manifest file to reference the news related activities. To do this, you need to add the following lines to your Android Manifest: 

 <activity android:name="com.scoreloop.client.android.ui.component.news.NewsHeaderActivity"/>
 <activity android:name="com.scoreloop.client.android.ui.component.news.NewsListActivity"/>
