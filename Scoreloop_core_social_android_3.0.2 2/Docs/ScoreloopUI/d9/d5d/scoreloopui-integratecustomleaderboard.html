Custom Leaderboards

If you wish, you may specify the mode and type of leaderboard that will be displayed by default when the Leaderboards view is launched.

Sample code snippet would be as follows: 

final Button leaderboardsButton = (Button) findViewById(R.id.button_leaderboards);
leaderboardsButton.setOnClickListener(new View.OnClickListener() {
@Override
public void onClick(final View v) {
   
   final Intent intent = new Intent(this, LeaderboardsScreenActivity.class);

   // optionally specify the mode you want the leaderboard to be opened in
   intent.putExtra(LeaderboardsScreenActivity.MODE, 1); 

   // optionally specify the leaderboard that will open by default 
   intent.putExtra(LeaderboardsScreenActivity.LEADERBOARD, LeaderboardsScreenActivity.LEADERBOARD_FRIENDS);

   startActivity(intent);
}
});
