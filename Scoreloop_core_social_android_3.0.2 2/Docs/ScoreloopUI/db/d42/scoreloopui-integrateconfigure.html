Configuring Your Game Properties

The following steps show how to properly configure your Scoreloop game application.

1. After you have registered your game on https://developer.scoreloop.com , navigate to the 'Game Properties' page on the side menu. Scroll down a bit and click on the 'Get Scoreloop Properties' link and save the file to a known location on your disk.

2. Open the scoreloop.properties file with a text editor. As part of basic integration, the following parameters are important: 

 # The game ID assigned to your game by Scoreloop (do not modify this)
 game.id = fe3f0ce6-01b8-4ba5-820a-9830ec6ed62b
 
 # The name of your game (provided by you when registering your game)
 game.name = Scoreloop Sample Game
 
 # The version of your game (game version should be 1.0 or above)
 game.version = 1.0 
 
 # Your game-specific currency code (do not modify this)
 currency.code = YNZ
 
 # The currency symbol as it will appear in the UI (you may modify this).
 currency.symbol = C
 
 # The singular and plural forms of your currency (you may modify these).
 currency.name.singular = Coin
 currency.name.plural = Coins

The ScoreloopUI properties section of this file enables you to configure/enable the additional features into your game. By default, these features are not enabled. How to enable these features are addressed in the respective advanced integration sections.

3. Once the scoreloop.properties file has been properly configured, you need to add it to the /assets folder of your game project.

4. You can also configure the sort order for scores in your game. By default, Scoreloop will sort scores in descending order. If you wish, you can opt to have scores in ascending order. To do this, go to https://developer.scoreloop.com and navigate to the "Score Configuration" page, shown below. The "Result" property refers to the game score. From the drop down menu, you can select whether the "Result" should be sorted in ascending or descending order. Ignore the "Level" and "Minor Result" properties. These are not taken into account by ScoreloopUI.

Once configuring your game properties is over, you will need to update the Android Manifest file. Please navigate to Updating the Android Manifest for more information on this.
