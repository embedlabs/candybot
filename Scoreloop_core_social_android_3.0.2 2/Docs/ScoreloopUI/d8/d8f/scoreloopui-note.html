A Note on Advanced Integration

  1. To enable advanced ScoreloopUI features you should have completed all steps from the basic integration guide.
  2. Depending on your game's user interface you may only want to have one entry point into Scoreloop. In that case please consider using the Scoreloop Entry Screen. Otherwise you have the option to link to the appropriate screens individually:
    • the Profile View,
    • the Challenges View,
    • the Awards View
  3. You may also consider linking directly to the Leaderboards view after gameplay finishes.
  4. You should ensure that each session of gameplay has a distinct start and endpoint. You should take care to ensure that an attempt to start a new gameplay session cannot occur while an existing session is underway. Gameplay can originate from the user (by starting a new game) or from ScoreloopUI (through a Scoreloop challenge). Every gameplay session must terminate properly with a call to:
    • ScoreloopManager.onGamePlayEnded(Double, Integer) or
    • ScoreloopManager.onGamePlayEnded(Score, Boolean). You may wish to implement a "Pause and Resume" function in your game. However, after a paused game has been resumed you must ensure that you call one of the onGamePlayEnded() methods listed above.
