Detailed Description

The AchievementController manages single instances of the Achievement class and is used to submit an Achievement to the server.

Once you submit an Achievement to the Scoreloop server, the user's balance will be updated by the reward specified before. Also, other uses will be able to see your unlock status only after submitting of Achievements.

To retrieve Achievement objects in the first place, use an AchievementsController.

Basic Usage:

To submit an Achievement to the server:

  1. Use the Achievement.needsSubmit() method of the Achievement class to check whether the achievement needs to be submitted to the server.
  2. If true, create an AchievementController instance or re-use an existing one.
  3. Call the AchievementController's setAchievement() method to set the Achievement.
  4. Call the AchievementController's submitAchievement() method.
  5. Await notification of success via delegate callbacks.
