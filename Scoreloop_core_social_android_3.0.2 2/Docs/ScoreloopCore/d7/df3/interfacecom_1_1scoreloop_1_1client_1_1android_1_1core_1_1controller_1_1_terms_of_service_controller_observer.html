void termsOfServiceControllerDidFinish (TermsOfServiceController controller, Boolean accepted)

Detailed Description

The TermsOfServiceControllerObserver class handles delegate callbacks when accepting or rejecting the TermsOfService. Whenever a player is about to establish a connection with Scoreloop for the first time, a Terms Of Service dialog will pop up. The user has the accept it so as to continue to use Scoreloop.

Callbacks will be triggered in the following circumstances:

  • Player accepts the TOS
  • Player rejects the TOS
