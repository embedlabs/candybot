Detailed Description

The GameItem class is used to model a virtual good in your Game GameItem content is uploaded by the game developer to the Scoreloop servers, and is then available for download by users. You can manage your GameItems from the Scoreloop developer console (https://developer.scoreloop.com).

See also:
GameItemController, GameItemsController
