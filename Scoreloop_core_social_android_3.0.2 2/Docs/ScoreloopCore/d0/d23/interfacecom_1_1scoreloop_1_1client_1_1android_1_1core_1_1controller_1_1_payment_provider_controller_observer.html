Detailed Description

The PaymentProviderControllerObserver class handles delegate callbacks when the checkout process has been initiated or started. Callbacks will be triggered in the following circumstances:

  • the checkout process is cancelled,
  • the checkout process failed,
  • the checkout process succeeded,
  • the checkout process finished with a pending payment.
See also:
PaymentProviderController, PaymentController, PaymentMethodsController
