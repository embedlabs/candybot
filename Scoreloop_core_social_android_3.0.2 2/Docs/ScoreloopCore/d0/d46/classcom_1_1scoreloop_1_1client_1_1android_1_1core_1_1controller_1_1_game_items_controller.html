Detailed Description

A data controller that allows access to lists of game items. The GameItemsController class is used to retrieve the list of all game items currently stored on the server. Game items refer to virtual goods that are made available to users of a game to download.

Game Items must be configured on the Scoreloop developer website https://developer.scoreloop.com.

Basic Usage:

  • Get an instance of the GameItemsController.
  • Request the list of game items using the loadGameItems() method.
  • Await notification of success via delegate callbacks.
  • After a successful server request, access the retrieved game items by calling the getGameItems() method.

GameItemController, GameItem, RequestController

void loadGameItems ( )

This method requests all available game items currently stored on the server. Retrieved items are loaded into the gameItems array which may be accessed by calling the getGameItems() method. The returned list includes free or paid items and also already owned items.
