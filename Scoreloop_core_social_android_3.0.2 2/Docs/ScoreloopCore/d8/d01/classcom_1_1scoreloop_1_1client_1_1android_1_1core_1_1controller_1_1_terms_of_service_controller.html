Detailed Description

A controller used to manage the terms of Service which the user has to accept to continue using the Scoreloop services.

Whenever a player is about to establish a connection with Scoreloop for the first time, a Terms Of Service dialog will pop up. The user has the accept it so as to continue to use Scoreloop. Please note that the Terms Of Service is per Game. If the User has accepted the Terms once for a particular game, the dialog will not be shown again for that game. However, he will have to accept it again in another game. However, if the user has rejected the Terms, he will not be allowed to continue interacting with Scoreloop. Game developers have to ensure that the players have accepted the Terms.
