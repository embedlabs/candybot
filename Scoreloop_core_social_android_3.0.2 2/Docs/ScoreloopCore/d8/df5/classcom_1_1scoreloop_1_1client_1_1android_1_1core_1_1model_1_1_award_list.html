Detailed Description

The collection of the Award objects available in the game.

Basic Usage:

  1. Get an instance of AchievementsContoller.
  2. Use the controller to request a list of achievements from the server.
  3. After a successful server request, use the controller to call AchievementsContoller.getAwardList(). This will return an instance of AwardList.
  4. Call either:
    • getAwards()
    • getAwardWithIdentifier() to access either a list of awards, or a single identified award, as required. (Use getAwardIdentifiers() to get the list of all AwardIdentifiers if needed).
