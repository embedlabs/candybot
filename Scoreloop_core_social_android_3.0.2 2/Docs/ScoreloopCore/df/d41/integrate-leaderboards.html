Leaderboard

The following table below gives an overview of the classes (including the purpose) associated with "Scores and Leaderboards".

ClassPurpose
Score
  • Model class for Scoreloop Score
ScoreController
  • Submitting a score to Scoreloop
ScoresController
  • Retrieve lists of scores from Scoreloop servers
  • filter lists of scores with respect to different criteria (e.g. wrt different game modes, buddy-lists, etc)

Basic Integration

Use case 1: Submitting a score

  1. Create/get the Score object to be submitted.
  2. Create an instance of ScoreController.
  3. Submit the score.
 //The Score to be submitted
 final Score score = new Score((double) scoreValue, null);
 
 //Other code will go here
 ...
 
 //Create the ScoreController instance
 final ScoreController myScoreController = new ScoreController(new ScoreSubmitObserver());
 
 //Use the controller to submit to the server
 myScoreController.submitScore(score);
 
 //It is a good idea to show a progress indicator here
 showProgressIndicator();

Use case 2: Retrieving scores from the server and displaying a leaderboard

  1. Create an instance of ScoresController.
  2. Set the SearchList for the controller.
  3. Set the mode for the controller, (the controller will only retrieve scores whose mode matches this value).
  4. Set the range length of scores that will be retrieved with each server request.
  5. Make the request to the server.
  6. Await a successful server response via the delegate callbacks.
  7. After a successful response, access the list of score objects using the controller's getScores() method.
 //Create an instance of the controller
 ScoresController myScoresController = new ScoresController(new myScoresControllerObserver());
 
 //Set the searchlist. Default is globalScoreSearchList
 myScoresController.setSearchList(SearchList.getBuddiesScoreSearchList());
 
 //Set the mode for the controller.
 myScoresController.setMode(2);
 
 //Set the range length of scores to be retrieved. Default is 25
 myScoresController.setRangeLength(20);
 
 //Make the request to the server
 myScoresController.loadRangeForUser(Session.getCurrentSession().getUser());

Await the successful call via the delegate callback methods, such as:

 //A standard delegate callback
 public void requestControllerDidReceiveResponse(RequestController aRequestController) {
 
 //Additional code here
 ...
 }

Once a successful server request has been made, then the list of score objects can be accessed.

 //Access the retrieved list
 List<Score> retrievedScores = myScoresController.getScores();
