Detailed Description

The Challenge class models all aspects of a Scoreloop challenge. See the Guide to Challenges for a more detailed explanation of a Scoreloop challenge.

Basic Usage:

Currently Challenge objects can be created in two ways.

Either:

  • Use the constructor below to create the challenge instance with a given stake,
  • then retrieve the current session object from the Client instance, and use it to call Session.setChallenge(), to set the challenge in the current session,
  • before setting the contender using setContender() below.

Or:

  • Use the createChallenge(Money, User) method of the ChallengeController class. This will automatically set the challenge in the current session and set the contender equal to be the current session user.
See also:
Guide Challenges, ChallengeController, ChallengesController
