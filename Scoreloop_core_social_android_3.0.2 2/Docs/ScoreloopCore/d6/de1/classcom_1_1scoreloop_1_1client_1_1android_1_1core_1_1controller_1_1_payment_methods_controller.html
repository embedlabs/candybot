Detailed Description

The PaymentMethodsController class is used for retrieving the available payment methods for a game item and the available prices for each of the options. Some of the payment options available are the following: credit card, pay pal, in-game currency etc.

void loadPaymentMethods ( )

Method to load the different payment methods available for a GameItem.

The game item for which the options need to be checked need to be set by calling setGameItem() on the controller. Once the payment methods are loaded, it may be accessed by calling getPaymentMethods().
