Detailed Description

The PaymentController class checks for the status of a given payment on the server.

Basic Usage:

  • Create an instance of the PaymentController.
  • Set the payment whose status needs to be checked by calling setPayment().
  • Call loadPayment() to load the latest status from the server.
  • Access the loaded payment by calling getPayment().
