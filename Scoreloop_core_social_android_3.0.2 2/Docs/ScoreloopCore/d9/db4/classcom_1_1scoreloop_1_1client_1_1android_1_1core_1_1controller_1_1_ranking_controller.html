Detailed Description

The RankingController retrieves the Ranking for either a User or a Score from the server.

The ranking refers to the position of either the user or score on a leaderboard.

Basic Usage:

  1. Set the searchlist for the RankingController using setSearchList(final SearchList).
  2. Request the ranking from the server using one of:
    • loadRankingForScore()
    • loadRankingForScoreResult()
    • loadRankingForUserInGameMode()
  3. Access the requested Ranking using getRanking()

Important usage note:

Do not modify a score object, or one of its sub-objects like the userContext, while one of the following asynchronous methods is in progress, as this might result in undefined results:

  • loadRankingForScore()
  • loadRankingForScoreResult()
  • loadRankingForUserInGameMode()

Note, that as of the 2.3 release, local scores search lists are not supported by this class.
