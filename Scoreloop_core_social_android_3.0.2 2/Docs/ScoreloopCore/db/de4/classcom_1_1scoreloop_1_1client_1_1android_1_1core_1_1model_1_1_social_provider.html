Detailed Description

The SocialProvider class models a social network. The social providers that are currently supported are:

  • Facebook
  • Twitter

Basic Usage:

To connect to a social provider, first:

  1. Call getSocialProviderForIdentifier() with the appropriate identifier in order to obtain the relevant social provider for the social network you want to participate in.
  2. Call isUserConnected(User) to see if the user is already connected to the social provider.
  3. If they are not connected, connect using an instance of SocialProviderController.
See also:
SocialProviderController, SocialProviderControllerObserver
