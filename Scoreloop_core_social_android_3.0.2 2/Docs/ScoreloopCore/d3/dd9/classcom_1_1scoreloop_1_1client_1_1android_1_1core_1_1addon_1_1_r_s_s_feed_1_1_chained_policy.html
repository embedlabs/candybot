 ChainedPolicy (final Policy...policies)

Detailed Description

The ChainedPolicy class is used to "chain" a number of sub-policies together.

The ON_START_POLICY , for example, is defined as a ChainedPolicy linking:

  • UNREAD_POLICY and
  • STICKY_POLICY together.
