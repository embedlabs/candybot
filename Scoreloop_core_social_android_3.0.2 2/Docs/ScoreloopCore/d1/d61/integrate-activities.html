Basic Integration

The following example demonstrates how to retrieve activities from the server.

Use case: Retrieving activities from the server

  1. Get an instance of ActivitiesController or re-use an existing one.
  2. Use the controller to make the server request using:
    • loadGameActivities(), to request all the activities for all users of the current game.
    • loadBuddyActivities(), to request activities for the buddies of the session user.
    • loadCommunityActivities(), to request all Scoreloop related activities (of all games and all users).
    • loadActivitiesForGame(), to request all activities in a game.
    • loadActivitiesForUser(), to request activities associated with a specific user.
  3. Await notification of success via delegate callbacks.
  4. After a successful server response, access the list of retrieved activities. Use getActivities() to access the list that was retrieved.
 //Create an instance of the controller.
 ActivitiesController myActivitiesController = new ActivitiesController(new myRequestControllerObserver());
 
 //Make the server request using one of the load methods. 
 myActivitiesController.loadBuddyActivities();

Await the successful call via the delegate callback methods, such as:

 //A standard delegate callback
 public void requestControllerDidReceiveResponse(RequestController aRequestController) {
 
 //Additional code here
 ...
 }

After notification of a successful server request, access the list of activities retrieved.

 //Access the list.
 List<Activity> = myActivitiesController.getActivities();
