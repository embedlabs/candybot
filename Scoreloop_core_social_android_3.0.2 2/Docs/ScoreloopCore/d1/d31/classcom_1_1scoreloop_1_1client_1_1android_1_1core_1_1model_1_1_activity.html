Detailed Description

The Activity class is used to model a Scoreloop-related activity for a user.

Activities will relate to one of the following Scoreloop events:

  • a Scoreloop Score,
  • a Scoreloop Challenge,
  • a Scoreloop Achievement.

Activity objects contain a message about the Scoreloop event the Activity relates to. An example message, (relating to a score-based Scoreloop event), might read: "Player001 has scored 50 points in the game SLDemoCore".

Basic Usage:

To retrieve the message associated with a given Activity, do the following:

  1. Get an instance of ActivitiesController.
  2. Use the controller to make the server request using:
    • loadGameActivities(), to request all the activities for all users of the current game.
    • loadBuddyActivities(), to request activities for the buddies of the session user.
    • loadCommunityActivities(), to request all Scoreloop related activities (of all games and all users).
    • loadActivitiesForGame(), to request all activities in a game.
    • loadActivitiesForUser(), to request activities associated with a specific user.
  3. Await a successful response via delegate callbacks.
  4. After a successful server request, use the controller to call getActivities() to retrieve the java.util.List of Activity objects.
  5. Select the desired Activity from the retrieved list.
  6. Call getMessage(), to retrieve the message of the Activity.
  7. Call getUser(), to retrieve the user associated with the Activity. Please note that you will have to use the UserController to resolve the Entity returned by getUser() into a proper User object.
