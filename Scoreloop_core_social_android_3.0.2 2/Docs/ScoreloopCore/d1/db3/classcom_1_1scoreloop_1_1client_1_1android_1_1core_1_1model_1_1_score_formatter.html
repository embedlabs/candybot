Detailed Description

Utility class for formatting game's score.

Following elements of a score can be displayed:

  • mode
  • level
  • result
  • minor result

Each of these elements may have a "Symbol" associated, i.e. a human readable description of the unit. (like collected "Gems", destroyed "Enemies" etc., "Difficulty" for the level).

Results may either use the numeric value or be converted as time using with ResultTimeConversion

Format string elements:

  • r : result
  • T : result as time with "ResultTimeConversion"
  • R : result symbol string ("ResultSymbol")
  • q : minor result
  • t : minor result as time with "ResultTimeConversion"
  • Q : minor result symbol string ("MinorResultSymbol")
  • l : level
  • L : level symbol string ("LevelSymbol")
  • m : mode
  • M : mode symbol string ("ModeSymbol")
  • S : separator string between level, mode and result parts

ResultTimeConversion only format:

  • h : hours
  • m : minutes
  • 0m : minutes, zero-padded to two digits
  • s : seconds
  • 0s : seconds, zero-padded to two digits
  • q : 1/100 seconds
  • 0q : 1/100 seconds, zero-padded to two digits

Constructor & Destructor Documentation

ScoreFormatter ( final String  format)

Parses a string describing custom formatting. The string has to be in form "key1"="value1";
"key2"="value2";
..."key_n"="value_n";. The string doesn't have to contain all format definitions. If a format key is not present in the string the default value for this key will be used:

"DefaultFormat"="%r%R %S %L %l";
"ResultTimeConversion"="%h:%0m:%0s";
"ScoresOnlyFormat"="%r%R";
"LevelOnlyFormat"="%L %l";
"ModeOnlyFormat"="%M %m";
"LevelAndModeFormat"="%L %l";
"ScoresAndLevelFormat"="%r%R %S %L %l";
"LevelSymbol"="Level";
"ModeSymbol"="Mode";
"ResultSymbol"="";
"MinorResultSymbol"="";
"Separator"="for";

Parameters:
formatformat description, if it is null only default values will be used
