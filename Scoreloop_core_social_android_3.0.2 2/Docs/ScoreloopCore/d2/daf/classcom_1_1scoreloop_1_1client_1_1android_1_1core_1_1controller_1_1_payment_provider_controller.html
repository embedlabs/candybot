Detailed Description

The PaymentProviderController class is used to manage the purchase of a given GameItem.

Basic Usage: The following steps need to be done:

  • Use a PaymentMethodsController to load the PaymentMethods for the GameItem.
  • Let the user choose one of the provided PaymentMethods.
  • Call PaymentProviderController.getPaymentProviderController() to get the PaymentProviderController for that PaymentMethod. _ Call checkout() on the PaymentProviderController to start the checkout process.
  • You will receive a callback on the PaymentObserver you provided.
