Detailed Description

A data controller used to download a specific item of DLC.

The GameItemController is used to download a specific virtual good from Scoreloop's Infrastructure. Game developers can offer Game Items to users for free, or purchasable via virtual in-game currency or real world money. Depending on how the game item has been defined on the server, a valid payment credential may be required before downloading can occur.

Basic Usage:

  • Get the list of available game items by calling loadGameItems() on the GameItemsController. This returns the list of free / paid and already owned items. Steps include:
    • Get an instance of the GameItemsController.
    • Request the list of game items using the loadGameItems() method.
    • Await notification of success via delegate callbacks.
    • After a successful server request, access the retrieved game items by calling the getGameItems() method.
  • Select the item to be download by calling setGameItem() and then call loadGameItem().
  • If the item selected from the list is a free item, the steps required to download the free item are:
    • First call the method submitOwnership().
    • Call loadGameItemDownloadUrl() which gives you the download url from where to download the game item.
    • Download the game item. Please note here that the download url that you recieve would be only valid for 5 mins and within that time, the game item needs to be downloaded.
  • If the item is not free and it has already been acquired. The steps involved are:
    • Call loadGameItemDownloadUrl() which gives you the download url from where to download the game item.
    • Download the game item.
  • If the game item needs to be purchased. The following steps need to be done:
    • Use a PaymentMethodsController to load the PaymentMethods for the GameItem
    • Let the user choose one of the provided PaymentMethods
    • call PaymentProviderController.getPaymentProviderController() to get the PaymentProviderController for that PaymentMethod
    • call checkout() on the PaymentProviderController to start the checkout process
    • You will receive a callback on the PaymentObserver you provided.

void loadGameItem ( )

This method makes a request to get the latest status of the GameItem that is to be downloaded. The item to be downloaded may be specified by setting the game Item by calling setGameItem().

void loadGameItemDownloadUrl ( )

This method requests for the download url from where the game item may be downloaded. Once the url is obtained, you may initiate a regular HTTP request to actually download the game item.

Please note here that the download url that you recieve would be only valid for 5 mins and within that time, the game item needs to be downloaded.

void submitOwnership ( )

Method to let the user have access to the game item. That is - it will mark the item (in this case the 'free' one) as one belonging to the player and the player may download it. If the item is a free item and if this method is not called, there is a possibility that loadGameItemDownloadUrl() will fail.
