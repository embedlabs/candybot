ScoreloopCoreDemoTypical - Developer Setup

This document describes the settings needed on developer.scoreloop.com for the Scoreloop features demonstrated in ScoreloopCoreDemoTypical.

To keep the Scoreloop settings in sync with your game, you should download the scoreloop.properties file from the Game Properties page after you have made changes. If you modified the list of Awards, you need to download and unpack the Awards Bundle file. Both of these resources have to be placed in the assets folder in your project.

Awards

Make sure to release your Awards list after setting it up. If you need to change one of the Awards, you need to un-release the list.

In-Game Currency

For general amusement the In-Game Currency is named Horses. Every new User will recieve 15 Horses to play for in Challenges.

Coin Packs

To allow a user to buy more Horses, you need to create Coin Packs
